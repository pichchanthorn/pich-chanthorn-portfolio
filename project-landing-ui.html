UI/UX & Front-End

About this Project

This Landing Page UI Collection is a set of practice designs focused on building clean, modern, and conversion-focused landing pages. Each design explores different layouts, hero sections, feature blocks, and call-to-action components.

I used this project to improve my skills in both UI design and front-end development. Some layouts were first created in Figma, then implemented using HTML and CSS to match the visual design as closely as possible.

These landing pages can be used as templates for future real-world projects such as SaaS products, portfolios, agency websites, or marketing campaigns.

Key Features

Multiple hero layouts with different text and image structures.
Use of modern UI patterns like cards, badges, and gradient backgrounds.
Feature sections that highlight key benefits and product features.
Testimonial and social proof blocks for building trust.
Strong call-to-action buttons designed to stand out visually.
Responsive layout for different screen sizes using clean CSS.

Key Achievements

1

Strengthened UI/UX sense by recreating modern landing page layouts from scratch.

2

Practiced translating Figma designs into pixel-perfect HTML & CSS implementations.

3

Built a reusable set of sections (hero, features, testimonials, CTAs) for future projects.
