About this Project

This portfolio website is designed to present my journey as an Information Technology student and aspiring full-stack developer. It focuses on clean layout, readable typography, and a professional dark theme with green highlights.

The site includes dedicated sections for skills, projects, education, certificates, and work experience. Each section is structured for clarity and smooth navigation on both desktop and mobile devices.

The goal of this project is to provide a single professional platform where I can showcase real-world work, track my growth, and connect with future employers or collaborators.

Key Features

Responsive layout for all devices
Sidebar navigation with active states
Modern dark UI with green accents
Multi-page structure for clear content
Project cards linking to case studies
Focus on web, UI/UX, database & AI tools
