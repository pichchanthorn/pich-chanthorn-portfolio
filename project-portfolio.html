About this Project

This portfolio website is designed to present my journey as an Information Technology student and aspiring full-stack developer. It focuses on clean layout, readable typography, and a professional dark theme with green highlights.

The site includes dedicated sections for my skills, projects, education, certificates, and work experience. Each section is structured to be easy to navigate and optimized for both desktop and mobile screens.

The goal of this project is to have a central place where I can showcase real-world work, track my growth over time, and share my contact information with future employers or collaborators.

Key Features

Responsive design for desktop, tablet, and mobile.
Sidebar navigation with smooth active state and hover effects.
Modern dark UI with green accent colors and soft shadows.
Separate pages for Skills, Projects, Certificates, Education, and Experience.
Clickable project cards that open detailed case-study pages.
Content focused on web development, UI/UX, database, and AI tools.

Key Achievements

1

Designed a consistent design system (colors, typography, spacing) that can be reused across future projects.

Practiced semantic HTML structure and clean reusable CSS for multi-page layouts.

3

Improved portfolio presentation by clearly organizing skills, projects, and achievements in one place.
