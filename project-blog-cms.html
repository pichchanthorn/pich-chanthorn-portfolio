Blog & CMS Platform

A content management interface for creating, editing, and publishing blog posts, designed with a clean layout for both writers and readers.

2024
Front-End Developer

About this Project

This Blog & CMS Platform is a practice project that simulates the core features of a simple content management system. It provides a layout where admins can manage posts and visitors can read articles comfortably.

The interface separates the public blog listing from the “admin style” controls used to create and edit posts. This helps me understand how dashboards and front-facing pages can share a design system but have different purposes.

Even though this version is front-end focused, it can later be extended with a real backend API, authentication, and a database to handle real content at scale.

Key Features

Create and edit blog posts with title, date, and content.
Display a list of recent posts on the main blog page.
Highlight featured articles or important updates.
Use simple categories or tags to group related posts.
Clean article layout for comfortable reading on any device.
Responsive UI that adapts to desktop, tablet, and mobile.
